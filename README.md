# UbicompLab-Pic24

PIC24F Starter Kit Authentication System - A 5-button capacitive touch interface with OLED display for user authentication.

## Features
- 5-button capacitive touch input (UP, RIGHT, DOWN, LEFT, CENTER mapped to digits 1-5)
- 128x64 OLED display with custom font rendering
- User registration and login system
- Modular code structure for easy customization

## Hardware
- PIC24F Starter Kit 1 board
- SH1101A OLED Display
- 5 capacitive touch pads
- RGB LEDs
